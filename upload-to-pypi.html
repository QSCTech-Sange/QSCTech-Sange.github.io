想要用pip install XXX 来安装你自己写的代码吗？
发表于2019-11-23|更新于2020-03-24|Python
|字数总计:1.1k|阅读时长:4分钟|阅读量:|评论数:

当我们通过pip install XXX来安装某个包的时候，总是感觉这个包特别的高达上，特别的正式。其实，将你自己的包上传到Pypi并没有那么困难。本篇教程就是简单几步教你将你的包上传到Pypi。

准备结构

首先你需要一个文件夹，这里起名叫package_example。在这个文件夹里建立一个setup.py，一个LICENSE，一个README.md，和一个文件夹。前面的三个文件都是文本文件，我们待会会逐一讲解。后面的文件夹我起名叫my_pkg，你可以按照你的喜好为它起名。在这个文件夹里，再新建一个__init__.py，也是一个空的文本文件。你的结构完成后应该如下所示。

而你自己写好的XXX.py文件，应该放在my_pkg文件夹里。

我将分文件介绍它们的作用。

init.py

当我们执行import XXX的时候，我们实际上就是调用了这个包的__init__.py函数。所以我们的__init__.py就是一个类似于头文件的东西。假如你在my_pkg上放了一个XXX.py文件，你想要导入包的时候就导入这个文件里的内容，那么你可以在__init__.py里这么写。
