你期待已久爱不释手的编程字体！

背景

提到最合适的编程字体，你会想到什么？熟悉的Consolas，苹果的Monaco，相信都是出场率最高的字体。

Cascadia

然而在今天，微软的Cascadia Code 横空出世，号称“最合适的编程字体”。

其有如下特点

  • 0,O,o可以被明显区分
  • 1,I,l,L 可以被明显区分
  • 等宽
  • 连字功能，可以将<=连成≤\le≤号。

虽然看起来很简单，却是直击了广大程序员的痛点。

下载安装

到这里下载最新的Cascadia.ttf。
Windows 用户直接打开安装即可。
Linux 用户需要放大/usr/share/font然后刷新字体缓存。有些麻烦但是你已经用Linux了相信你会的!

注意在IDE或者编辑器中需要打开连字才可以享受连字效果。

我已经将shell，所有的编辑器和IDE都换成这个字体啦！真是爱不释手！
